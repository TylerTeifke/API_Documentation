API Documentation

This website will hold the documentation on our APIs

lightSource functions

setColor(r,g,b)

This will allow the user to set the color of the light to whatever they want

incRedBy(delta)

will allow the user to increment the red value of a light by a set amount

incGreenBy(delta)

will allow the user to increment the green value of a light by a set amount

incBlueBy(delta)

will allow the user to increment the blue value of a light by a set amount

getColor()

will return the red, green, blue, and alpha values of the light

getXform()

will return the transform object of the light, which will allow the user to control where the light is placed

setLightRange(val)

will allow the user to control how much area the light illuminates

incLightRange(delta)

will allow the user to increment the light range by a specific amount

getLightRange()

will return the range of the light object

setBrightness(val)

will allow the user to control how bright the light is

incBrightness(delta)

will allow the user to increment the brightness of a light by a specific amount

getBrightness()

will return the brightness of the light object

turnOnOrOff()

will turn the light on if it is off or will turn it off if it is on

getIsOn()

will return true if the light is on and false if the light is off

lightRenderable functions

constructor(texture)

will build an object that is able to interact with light by attaching light objects to the inputted texture

draw(camera)

will draw the object and all of the lights attached to it

getLightSource(index)

will return the light source corresponding to the above index

addLightSource(light)

will add the above light object to the array of lights that affect this object

removeLightSource(index)

will remove the light object that corresponds to the index from the array of lights that affect this object
